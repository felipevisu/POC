# Provador de Marcas

Upload a logo, pick a few surfaces (business cards, polos, signage, packaging…), and see your brand applied to all of them in parallel — fotorealistic mockups generated by `google/gemini-3.1-flash-image-preview` via OpenRouter.

![Screenshot](./screenshot.png)

## Run it

```bash
npm install
echo "OPENROUTER_API_KEY=sk-or-..." > .env.local
npm run dev
```

Open http://localhost:3000.

## How it works

- `app/page.tsx` — the editorial-style UI: logo upload, surface picker, per-surface prompt overrides, aspect-ratio / size controls.
- `app/api/generate/route.ts` — POSTs the logo (as a data URL) plus prompt to OpenRouter and returns the generated image URL.
- `lib/applications.ts` — the catalog of surfaces with default prompts and suggested aspect ratios.

Each selected surface is generated as an independent request, so the gallery fills in as results come back.
