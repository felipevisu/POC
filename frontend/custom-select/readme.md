A simple book search app with pagination.

## Setup

### Backend

```bash
cd backend
npm install
npm run dev
```

Server runs on http://localhost:5000

### Frontend

```bash
cd frontend
npm install
npm run dev
```

App runs on http://localhost:3000

## Features

- Search books by name
- Load more books (20 at a time)
- 101 books total
