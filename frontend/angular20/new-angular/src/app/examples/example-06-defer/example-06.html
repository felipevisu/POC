Angular 20 — Control flow & @defer

Built-in template syntax (v17+): @if / @for / @switch, and @defer for lazy-loaded blocks. No NgIf / NgFor imports.

@switch + @if

{{ status() }}
@switch (status()) { @case ('idle') {

⏸ Idle — nothing running.

} @case ('active') {

▶ Active — work in progress.

} @case ('done') {

✔ Done.

} }

@defer (on viewport)

Scroll the panel into view — its chunk loads on demand. Placeholder shows first.

@defer (on viewport) { } @placeholder {
Comments not loaded yet (placeholder).
} @loading (minimum 300ms) {
Loading comments chunk…
} @error {
Failed to load.
