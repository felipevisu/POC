Angular 20 — Chrome DevTools integration

enableProfiling() makes Angular publish change-detection and rendering timings to Chrome's Performance panel as a dedicated Angular track — no extension needed.

  1. Click Enable profiling.
  2. Open DevTools → Performance → click record (●).
  3. Click Churn 120 frames, wait ~2s, stop the recording.
  4. Find the Angular track — entries for change detection & render per frame.
